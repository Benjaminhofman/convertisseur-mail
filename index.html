Créer des variables et générer plusieurs mails


    

      
1 · Créer une variable

      
Dans le texte de votre mail, sélectionnez à la souris un mot ou un passage qui doit changer selon le destinataire (par exemple le nom du client). Un bouton « Transformer en variable » apparaît : cliquez dessus, donnez un nom à la variable (ex. « Nom du client »), validez. Le texte sélectionné devient une variable réutilisable — vous n'avez plus besoin de retenir de syntaxe particulière.

      
"Bonjour Monsieur Martin," → sélection de "Monsieur Martin" → "Bonjour {{ Nom du client }},"

    


    

      
2 · Réutiliser la même variable plusieurs fois

      
Si le même élément apparaît ailleurs dans votre texte (ex. le nom du client cité une seconde fois), sélectionnez-le aussi et choisissez « Utiliser une variable existante » au lieu d'en créer une nouvelle. Les deux emplacements se mettront à jour ensemble.

    


    

      
3 · Générer un mail par destinataire

      
Une fois vos variables créées, la section « 3 · Destinataires » apparaît. Ajoutez-y une ligne par destinataire (manuellement ou en collant depuis Excel), une colonne par variable. Cliquez sur « Générer les mails » : vous pouvez alors naviguer entre les destinataires et copier, pour chacun, un mail entièrement personnalisé.

    


    
Astuce : modifiez la valeur d'une variable dans le panneau « Variables du mail » pour mettre à jour tous les mails déjà générés en un instant, sans retoucher le texte.
